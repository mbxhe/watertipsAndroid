# WaterReminder 사용자 안내서

## 개요

WaterReminder는 하루 물 섭취 목표를 설정하고, 지정한 시간 간격과 시간대에 알림을 보내며, 200ml/300ml 버튼으로 손쉽게 섭취량을 기록하는 간단한 앱입니다. 진행률은 목표 대비 오늘 섭취량으로 표시됩니다.

## 주요 기능

- 빠른 기록: 200ml·300ml 버튼을 눌러 오늘 섭취량을 누적.
- 진행률 표시: 목표 대비 섭취량을 진행률 바와 텍스트로 확인.
- 맞춤 알림: 하루 목표, 알림 간격, 시작/종료 시간을 설정(간격 최소 15분).
- 백그라운드 알림: WorkManager로 주기적 로컬 알림 전송(Android 13+ 알림 권한 필요).

## 설치 및 실행

1. Android Studio에서 저장소 루트 `watertipsAndroid`를 엽니다.
2. Gradle 동기화 후 실제 기기 또는 에뮬레이터(API 26+ 권장)를 준비합니다.
3. `app` 모듈을 실행해 화면과 알림을 확인합니다.

## 사용 방법

1. 기본값은 하루 목표 2000ml, 알림 간격 60분, 시간대 08:00-22:00입니다.
2. 설정 영역에 목표(ml), 간격(분), 시작/종료 시간을 입력하고 “저장”을 눌러 설정을 반영하고 알림 주기를 갱신합니다.
3. 물을 마실 때마다 200ml 또는 300ml 버튼을 눌러 기록하면 진행률이 즉시 갱신됩니다.
4. 알림을 탭하면 앱의 메인 화면으로 이동합니다.

## 권한 및 데이터

- 알림 권한: Android 13 이상은 최초 실행 시 알림 권한을 허용해야 알림을 받을 수 있습니다.
- 데이터 저장: 섭취 기록은 로컬 Room DB, 설정은 SharedPreferences에 저장되며, 앱 삭제 시 함께 제거됩니다.

## 자주 묻는 질문

- 알림이 오지 않아요: 알림 권한이 허용되었는지, 현재 시간이 설정한 시작/종료 범위에 있는지 확인하세요.
- 간격 오류: 알림 간격은 15분 이상이어야 하며, 정수로 입력해야 합니다.
