# WaterReminder 使用指南

## 概述

WaterReminder 是一个简单的喝水提醒与记录应用。用户可以设置每日目标、水分提醒间隔与时间窗口，并通过快捷按钮快速记录 200ml 或 300ml 的饮水量，应用会显示当前进度并在指定时间段内推送提醒。

## 主要功能

- 快速记录：点击 200ml / 300ml 按钮累加今日饮水量。
- 进度展示：以进度条显示当日已喝水量与目标值。
- 自定义提醒：设置每日目标、提醒间隔、开始与结束时间；间隔至少 15 分钟。
- 后台提醒：使用 WorkManager 按设定间隔触发本地通知（Android 13+ 需要通知权限）。

## 安装与运行

1. 使用 Android Studio 打开仓库根目录 `watertipsAndroid`。
2. 同步 Gradle 并连接真机或启动模拟器（API 26+ 推荐）。
3. 运行 `app` 模块，即可查看主界面并测试提醒。

## 使用步骤

1. 首次进入时，默认每日目标 2000ml，提醒间隔 60 分钟，时间窗口 08:00-22:00。
2. 在设置区输入新的目标（ml）、提醒间隔（分钟）、开始和结束时间，点击“保存”更新设置并重新安排提醒。
3. 每次饮水后点击 200ml 或 300ml 按钮记录，进度条会同步更新。
4. 提醒通知点击后会返回应用主界面。

## 权限与数据

- 通知：Android 13+ 首次启动会请求通知权限，否则无法收到提醒。
- 数据存储：饮水记录保存在本地 Room 数据库；设置保存在 SharedPreferences，卸载后会清除。

## 常见问题

- 未收到提醒：确认通知权限已允许，且当前时间在设置的开始与结束时间之间。
- 间隔太短报错：提醒间隔需至少 15 分钟，确保输入格式为整数。
